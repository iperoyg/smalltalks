# SmallTalks Detector

**_A simple regex solution for many portuguese smalltalks_**


## SmallTalks types
- Confirmation / Confirmação
- Confusion / Confus
- Consent / Anuência
- Disagreement / Discordância
- Greeting / Saudação
- GoodMorning, GoodEvening, GoodNight / Bom dia, Boa tarde, Boa noite
- Negation / Negação
- Surprise / Supresa
